## Production-Ready Boilerplate Suite

A set of **5 open-source template repositories** — each production-ready with authentication, database, email, UI components, payments (opt-in), Docker, and CI/CD. Zero external account dependencies.

| Template | Stack | Description |
|----------|-------|-------------|
| [nuxt-boilerplate](https://github.com/faizkhairi/nuxt-boilerplate) | Nuxt 3, Vue 3, Prisma, Shadcn-vue | SSR web app with Turborepo monorepo |
| [next-boilerplate](https://github.com/faizkhairi/next-boilerplate) | Next.js 15, React, Prisma, Shadcn | SSR web app with React Email |
| [laravel-boilerplate](https://github.com/faizkhairi/laravel-boilerplate) | Laravel 12, Inertia.js, Vue 3 | Full-stack with RBAC, audit logging, Stripe |
| [vue-springboot-boilerplate](https://github.com/faizkhairi/vue-springboot-boilerplate) | Spring Boot 3.3, Vue 3, Java 21 | Enterprise full-stack with OpenAPI + JWT |
| [expo-boilerplate](https://github.com/faizkhairi/expo-boilerplate) | Expo SDK 54, React Native | Cross-platform mobile with EAS Build |

**Start a new project in seconds:**

```bash
gh repo create my-app --template faizkhairi/nuxt-boilerplate --private --clone
```

Each template includes: authentication flow, email templates (Mailpit for dev), database migrations, Docker Compose, GitHub Actions CI/CD, and deployment configuration.
